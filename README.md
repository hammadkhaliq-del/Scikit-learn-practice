# Scikit-learn Mini Projects

This repository contains **12 hands-on mini projects** that I am building while learning **scikit-learn**. Each task focuses on a specific machine learning concept and is designed to reinforce both the theory and practical implementation of the library.

The goal is to understand **how and why** each algorithm or technique works rather than simply using it.

## Project Roadmap

| Task    | Dataset            | Focus                                |
| ------- | ------------------ | ------------------------------------ |
| Task 1  | California Housing | Core API + Regression                |
| Task 2  | Titanic            | Data Preprocessing                   |
| Task 3  | Titanic (Proper)   | Pipelines + ColumnTransformer        |
| Task 4  | House Prices       | Linear Regression vs Ridge vs Lasso  |
| Task 5  | Titanic            | Logistic Regression                  |
| Task 6  | Breast Cancer      | K-Nearest Neighbors (KNN)            |
| Task 7  | Wine               | Decision Trees                       |
| Task 8  | Digits             | Random Forest + Gradient Boosting    |
| Task 9  | Titanic            | Cross Validation                     |
| Task 10 | Titanic            | GridSearchCV (Hyperparameter Tuning) |
| Task 11 | Mall Customers     | K-Means Clustering                   |
| Task 12 | Breast Cancer      | Principal Component Analysis (PCA)   |

## Objectives

* Learn the scikit-learn API through practical implementation.
* Build intuition for common machine learning algorithms.
* Practice preprocessing, feature engineering, model evaluation, and hyperparameter tuning.
* Understand when to use different algorithms and techniques.
* Develop a solid foundation before moving on to more advanced machine learning and deep learning topics.

Each task is implemented independently, allowing concepts to be studied and revisited without relying on previous projects.
